# 【标题】当城市悄悄变空：东北人口下坡路上的一堂课
【副标题】人口的曲线在下滑，真正被改变的，是一座座城市里老师、父母和孩子的日常生活。
【选题说明】本文以国家统计局公布的 2024 年全国人口最新数据及相关研究为基础，结合东北长期人口变动趋势，从人口结构视角观察老工业基地的收缩与转型。
【作者】刘继强，黑龙江人，政治经济学博士，新闻评论员。
> 当城市悄悄变空：东北人口下坡路上的一堂课

人口的曲线在下滑，真正被改变的，是一座座城市里老师、父母和孩子的日常生活。

傍晚最后一声下课铃响起，操场上只剩下稀稀拉拉的几个孩子。
小王站在办公室的窗边，看着空下来的跑道，有点恍惚。

十年前，他刚分配到这所中学时，操场是要排队才能上去的——一个班五六十个学生，体育课一上，整片操场都是吵闹声。后来班额变小了，学校合并了两个教学班，隔壁那所小学也被并入镇中心小学，原来的教室成了仓库。再后来，他送走了一届又一届学生，越来越多的人考上大学后再也没回来。

“回来的，往往是最不想走的那一批；真正想走的，大概也不打算回来了。”
这是小王最近几年最深的感受。

如果只是停留在这样的感叹上，“东北在变空”“孩子少了”，很容易滑向一种情绪化的判断：年轻人太现实，老家留不住人，大家都在逃离。可是，对一座城市来说，人口变化不是几张冷清街景那么简单，它是可以被清清楚楚测量出来的。

过去二十年里，东北很多城市的人口持续净流出，按照一些公开的统计数据，在某些地级市，十年间常住人口减少了两成。县城一年出生的孩子，已经填不满当年的一个年级；农村人口快速减少，留下来的多是老人和孩子，“空心村”不再只是比喻，而是地图上越来越多的空白点。

对教育系统来说，这意味着什么？
意味着一边是撤点并校、班额下降，另一边是无形的紧张：曾经短缺的教师岗位，可能在十年内变成“冗员”；县里的孩子越来越少，优质教师却依然在往省城、大城市集中。像小王这样的中年老师，站在讲台上，隐约能感到一种矛盾——他手里握着的是一份正在“缩水”的事业：不是他的错，却实实在在发生在他身上。

要想不被情绪裹挟，就得把这座收缩中的小城，放进更大的坐标里。

第一重坐标，是全国。
人口学者已经反复提醒：东北今天正在经历的——低生育率、人口负增长、城市收缩、农村空心化——并不是一块“特殊病灶”，而是整个国家很可能在未来十到二十年陆续会遇到的现实，只不过东北来得更早、更猛一些。与其把东北当成“掉队者”，不如说，它是被时间推到前面的“先行区”。

第二重坐标，是世界。
在日本本州北部，在德国鲁尔区，在美国底特律，人们也曾经讨论过类似的问题：当制造业的辉煌退潮，人口开始减少，年轻人把目光转向更远的地方，老城市该如何自处？
东北不是宇宙孤例，而是全球老工业基地共同面对的一道题：当年轻人不再把工厂和老城视作未来，谁来维持这里的学校、医院、公交和供暖？谁来守住那些曾经支撑过国家工业化的土地和街区？

人口并不只是冷冰冰的数量，它通过一条条“看不见的链条”改变着公共生活的每个角落：

人少了，地方税收跟着变少，财政吃紧，学校合并、医院瘦身、公交线路缩减，最先感受到变化的，往往是偏远乡镇；

老龄化加速，养老金和医保支出不断攀升，年轻一代缴费压力加大，工作与养家的焦虑被叠加在一起；

农村空心化，土地撂荒的风险上升，农业靠谁来种？粮食安全不再是抽象的“国家大局”，而是一个个被留在村里的老人还种不种地的问题。

在这样一条因果链上，我们既可以看到风险，也可以看到被迫出现的机会。

当可用劳动力越来越少，粗放式发展再难为继，“用更多的人换取更多产出”的模式撑不下去，自动化、数字化、现代农业不再是“选修课”，而是生活本身的逻辑。
当养老和健康需求迅速膨胀，“银发经济”、社区养老、医养结合也不再只是会议里的词，而是可能撑起许多年轻人就业的新行业。
人口下坡路上，逼着我们去思考：用什么样的方式，安排好每一个阶段的生命，同时也安排好这片土地的未来。

“胸中有正义”，不是谁嗓门大就算正义，而是愿意站在哪些人的处境上思考问题。

在人口这道题上，至少有三组人，值得被刻意看见。

第一组，是被“双重挤压”的年轻人。
一边是“多生一点”“留在家乡”的期待，一边是房价、教育成本和就业不确定性堆成的现实高墙。当舆论习惯用“责任感”“爱不爱家乡”来拷问他们的时候，很少有人认真回答过一个朴素的问题：当地究竟能不能给出一份体面的工作、一种可预期的公共服务？
如果一个地区连基本的公共教育、医疗与养老都难以稳定提供，“留下来”便很难成为一句有说服力的劝告。

第二组，是默默承担风险的老年人。
当我们习惯用“养老负担”“财政压力”这样的词汇时，很容易不自觉地把老人当成一笔账、一项支出。
但这些数字背后，是几十年的工龄、农龄，是在国企、在林场、在农田里撑起这一地区的那一代人。他们正在变老，他们需要被照顾，更需要被尊重。
真正有温度的讨论，既要承认养老体系面临的压力，也要坚持一个底线：不能把老年人单纯当成被计算的对象，而要努力让他们成为参与者——他们有经验、有时间，也完全可能成为社区照护、代际教育、非全日制工作的一部分，是一种“第二次人口红利”。

第三组，是正在被悄悄边缘化的城乡孩子。
城市收缩和农村空心化叠加，最先被调整的往往是教育和医疗：乡镇小学撤并、寄宿制学校增加，小医院、卫生院被整合。
当我们在政策文本中谈“优化资源配置”的时候，很少有人去想象这样一幕：天刚蒙蒙亮，冬天的风把国道吹得呼呼作响，一个背着书包的小学生在一辆破旧的校车上打瞌睡，他要在这样的清晨，颠簸一个多小时，去上一个“被优化出来”的学校。
资源配置的每一次调整，都会落在一个具体孩子的脚步上——这是人口统计表里看不到的部分。

所谓“正义”，也许就是在写稿之前，先提醒自己：别只替那些有话筒、有渠道的人说话，也替那些没有声音的人留一个位置。

最后，是“心头有温暖”。

这不意味着粉饰太平，而是即便看清了局势的严峻，也仍愿意相信改善的可能，并尽力指出一些方向。

对年轻人来说，我们可以少一点道德指责，多一点制度上的诚意：
不是笼统地喊“留下来”，而是在就业、住房、子女教育上做出看得见的改变——让留在家乡工作，不是“无奈之选”，而是一种可以自信说出口的生活方案。

对老年人来说，我们可以尝试从“负担思维”走向“资源思维”：
在保证基本养老与医疗的前提下，鼓励那些有意愿、有能力的老年人参与社区服务、志愿活动和适度劳动，让他们不仅被照顾，更被需要，被看见。

对农村和小城镇来说，我们可以改变“任其凋零”的想象：
通过土地制度改革、现代农业和数字基础设施，让一些原本“留不住人”的地方，变成真正适合生活的“宜居小镇”；
让那些选择留下耕地、守护黑土地的人，靠技术升级获得体面的收入，而不只是靠牺牲青春和容忍差距。

写到这里，我们或许能重新理解那句看似冰冷的话：
人口的账，最后都要算在一个个家庭身上。

正因为如此，当我们讨论人口、财政、养老、教育的时候，更应该对数字负责，也对人心负责。

如果有一天，我们在谈论东北、谈论整个人口下坡路时，能做到——
背后有扎实的事实，眼里有更大的世界，胸中有对弱者的偏爱，心头有对每个普通人的体谅；
那么，即便结论仍然严峻，我们传递出去的，就不只是焦虑，还有一种：

在看清现实之后，仍愿意相互托付、共同寻找出路的温和与勇气。

只要这份温和与勇气还在，东北的故事，就远远没有走到写下句号的时候。

第二个版本

标题：
当城市悄悄变空：在人口下坡路上，谁能发出自己的声音？

副标题：
人口是一条下滑的曲线，真正被改变的，却是那些从来没有在决策桌上坐过的人。

傍晚最后一声下课铃响起，操场上只剩下零星几声呼喊。
小王站在办公室的窗边，看着空下来的跑道，有点恍惚。

十年前，他刚分配到这所中学时，操场是要排队才能上去的——
一个班五六十个学生，体育课一上，整片操场都是吵闹声。
后来班额变小了，学校合并了两个教学班，隔壁那所小学也被并入镇中心小学，原来的教室成了仓库。再后来，他送走了一届又一届学生，越来越多的人考上大学后再也没回来。

“回来的，往往是最不想走的那一批；真正想走的，大概也不打算回来了。”
这是小王最近几年最深的感受。

如果只停留在这样的感叹上——“东北在变空”“孩子少了”——我们很容易把矛头指向年轻人：嫌他们太现实，不愿意“吃苦”“守住家乡”。
但人口变化不是朋友圈里的几张冷清街景，它是可以被清清楚楚测量出来的；更重要的是，它背后牵着一整套制度性的选择：城市怎么规划，学校怎么撤并，土地怎么流转，公共服务为谁而供。

一

过去二十年里，东北很多城市的人口持续净流出。一些公开数据表明：在部分地级市，十年间常住人口减少了两成。某些县城一年出生的孩子，只够填满一两个班；农村人口不断减少，留下来的多是老人和孩子，“空心村”不再只是比喻，而是地图上越来越多的空白点。

对教育系统来说，这意味着什么？
意味着一边是撤点并校、寄宿制扩张，一边是无形但持续的紧张：曾经短缺的教师岗位，可能在十年内变成“冗员”；县里的孩子越来越少，优质教师却依然在向省城、大城市集中。像小王这样的中年教师，站在讲台上，隐约能感到一种矛盾——他手里握着的是一份正在“缩水”的事业：不是他的错，却实实在在发生在他身上。

在更宏观的统计表里，这个区域的人口结构也在发生深刻的改变：
劳动年龄人口逐年缩水，老年人占比不断攀升，新出生人口徘徊在一个偏低的平台上。
这些数字看上去抽象，却会沿着一条条“看不见的链条”，改变每一个人的日常生活：
人口减少 → 税收变少 → 地方财政吃紧 → 学校合并、医院瘦身、公交线路缩减；
老龄化加速 → 养老金和医保支出不断攀升 → 年轻一代缴费压力加大，工作与养家的焦虑被叠加在一起；
农村空心化 → 耕地撂荒风险上升 → 粮食和土地，变成需要用更高成本去守住的底线。

这条链条的每一个环节，都是制度决定资源怎么分配，却很少询问普通人“你愿不愿意”。

二

把这座悄悄变空的小城放在更大的坐标里，我们会看到另一层结构性的紧张——强势的政府，和弱势的群众。

在很多重大决策中，政府有极强的动员能力：
几十年前，一纸计划生育政策，改变了无数家庭的命运；今天，为了鼓励生育，又可以在文件里提出“优化生育支持政策”，从产假、补贴到购房落户，迅速设计出一整套激励。
当城市要扩张的时候，可以集中力量修高架、建新区、搞开发区；当财力吃紧、人口外流的时候，也可以迅速做出“撤点并校”“整合资源”的决定。

在这些宏大的动作背后，普通人往往处在一个相当被动的位置：
当年被动接受“一对夫妇只生一个”的，是普通家庭；
如今被动面对“想生也不敢生”“生了也养不起”的，仍然是普通家庭。
当年响应“上山下乡”的，是一代年轻人；今天被要求“支持家乡建设”的，还是一代又一代年轻人。
政策可以快速转向，人口结构却无法立刻掉头，代价往往由个体默默承担。

我们很少正面承认这一点：
在许多人口与区域政策的设计中，“国家”和“地方”是重要的主体，“市场”和“资本”也会占据相当位置；真正最受影响的那些人——农民、老师、小城公务员、普通工人——却几乎没有机会坐到桌边，讨论他们的命运。

他们是统计表里的“人口”，却并不总是政治意义上的“公民”。

三

在这套结构里，农村人的处境尤为值得反思。

过去很长一段时间，乡村被视作城市化的“后备池”——
它源源不断向城市输送劳动力、土地和粮食，却并不总能等来同等质量的教育、医疗和养老服务。
县城集中办学、乡村撤点并校，在很多地方是出于改善教学条件的初衷；
但在操作层面，乡村学校、乡村卫生院、乡镇文体站最先被“优化”，反而被保留下来的，是对土地和资源的持续索取。

一个典型的画面是这样的：
远方的文件提出，要发展现代农业、推动土地适度规模经营、提高粮食产量；
村庄里的真实情形却是：年轻人走了，剩下的老人还在种地，却不敢轻易流转承包权——那是他们对未来生活仅存的某种安全感。
当他们被劝说“把地流转出去，自己去享受晚年”，很少有人认真回答另一个问题：
“如果我不再种地，我到底靠什么体面地活着？”

这不仅仅是经济问题，更是尊严问题。
在许多宏大的发展叙事里，乡村似乎天然是“被发展”的对象，是城市的供给地、缓冲区、试验场，却很少被平等地承认为：这里的每一个人，也是有完整生活、复杂情感和独立判断的公民。
当我们习惯于从“产量”“效率”“规模”来衡量乡村时，往往就很难从“体面”“尊严”“选择权”的角度，去理解那位不愿离开土屋的老人、那位宁愿留在县城领一份低工资也不去大城市的年轻人。

四

历史的洪流，总是以宏大叙事出现：工业化、城镇化、人口转变、老龄社会……
但真正被洪流推着走的，往往是不发声的人。

那是一对在上世纪执行计划生育时被“做通思想工作”的夫妻，如今在养老体系的缝隙里苦苦支撑；
是一位在资源型城市干了一辈子重体力的工人，中年下岗后靠零工支撑一家老小，却在统计表里只是一行“劳动力转移”；
是一位乡村小学的代课老师，没有编制，几十年如一日守着空下去的教室，最后被一句“理顺体制”轻描淡写地带过。

在舆论场上，我们更容易听到的是那些“代表性声音”：专家、官员、企业家、意见领袖。
他们构成了公共讨论的重要部分，但如果只有他们，整个公共空间就像一间只开了一半灯的屋子——
那些在灰暗角落里的脸、那些不好意思开口的委屈、那些不擅长表达的悲喜，会被悄悄抹平。

要让反思真正有力量，可能就要从这里开始：
承认制度在分配资源、塑造命运方面的强大能力，也承认这种能力需要被监督、需要听见来自底层的迟疑和反对；
承认国家有权为未来做规划，也承认个体有权对自己的人生做出不一样的选择。

五

说到这里，“心头有温暖”并不是告诉自己“别想太多”，相反，它要求我们在看清结构性问题之后，仍然相信改善的可能，并愿意为这种可能留下空间。

对年轻人，我们可以少一些道德化的指责，多一点制度上的诚意：
不是简单地喊“留下来”“多生孩子”，而是在就业、住房、托育、教育上拿出看得见的改变——
让愿意留在家乡的人，不至于总是被生活推到悬崖边缘；
让选择离开的，也不用背负“背叛”的道德负担。

对老年人，我们可以从“负担思维”转向“同行思维”：
在保障基本养老和医疗的前提下，尊重他们继续劳动与参与的意愿，把他们视为经验与记忆的承载者，而不仅是需要被“安置”的一代人。
在政策语言中，少用一点“负担”“压力”这样的词，多承认一些“贡献”“权利”。

对农村和小城镇，我们可以尝试打破“注定凋零”的想象：
通过更公平的财政体制和公共服务，让一部分地方真正具备“留下来的理由”；
通过土地制度、集体经济和现代农业的创新，让农民不必在“离开”和“贫穷”之间二选一，而是在“留下”和“体面生活”之间寻找平衡。

最后，我们也许可以重新理解那句看似冷冰冰的话：
人口的账，最后都要算在一个个家庭身上。

正因为如此，当我们讨论人口、财政、养老、教育的时候，更应该：
既对数字负责，也对人心负责；既看到国家和城市的宏大叙事，也听见那些被时代裹挟、却很少被问“你怎么看”的小小声音。

如果有一天，我们在谈论东北、谈论整个人口下坡路时，能够做到——
既看见强势制度带来的效率，也敢于追问它对个体自由与尊严的代价；
既承认历史洪流的不可逆转，也愿意俯身聆听那些被洪流拍打的人；

那么，即便前路仍然艰难，我们传递出去的，就不只是焦虑，还有一种：
在冷静的反思之后，仍然愿意为更体面、更公平的生活留出可能的温柔和勇气。

只要这份勇气还在，东北的故事，就远远没有走到写下句号的时候。

这是一位生于变革年代、植根基层经验、受训于社会科学、投身于公共写作的“观察者-思想家-倡导者”。他/她像一个冷静而温暖的“社会医生”，既诊断结构性的病灶，也呵护个体的尊严。在舆论场趋向两极分化的时代，他/她坚持一种复杂、系统且充满同理心的叙述，试图弥合宏观与微观、制度与人性、效率与公平之间的裂痕。他/她的作品，是写给转型中国的一份份深度病理报告，同时也是一封封写给普通人的情书与声援信。
